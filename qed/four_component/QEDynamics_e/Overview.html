Overview

 

QEDynamics(4 component version) solves time differential equation of density matrix. The physical quantities such as charge density and stress tensor density can be computed from the density matrix and they are recorded in the output file.


The overview of the program is described as the figure on the left. This control structure is coded in simQED.f90. Below, we explain each part briefly.

[Read input file]

Read information from the input file (named as params.ini in our sample) which is passed to the executable file as the argument. Information such as the location of Dirac output files and computational setups are given.


[Read Dirac output data]

Read the output (mainly the coefficients of the orbitals) of Dirac code which is a publicly available code for relativistic electronic structure computation in 4 component. They are used for computing physical quantities and molecular integrals in the following. 


[Initialize density matrix]

As default, it starts from the ground state of Dirac-Hartree-Fock calculation. setting.


[Compute initial physical quantities]

Using the initial density matrix as set above, physical quantities such as electronic charge density at some positions (whose coordinates are hard-coded just before [Read input file] part. ) are computed.


[Compute molecular integrals]

Molecular integrals which will be used in the differential equations of the density matrix are computed using the Dirac output data. Integrals such as kinetic energy integral and two-electron integrals are computed by integration of gaussian functions. As for the two-electron integrals, those with larger values than the threshold are save on the disk at the first computation and when we use the same Dirac output data, we can use the saved two-electron integrals.


[Compute derivative of density matrix]

The time derivative of the density matrix is computed based on the equation of motion of the Rigged QED theory. In the present version, we employ Born-Oppenheimer approximation and only the density matrix of electrons is considered. Also, vector potential part of the photon field is neglected.


[Update density matrix]

In the present version, time evolution is solved by the Euler method.


[Compute and output physical quantities]

The physical quantities are computed every time step which is defined in the input file and they are written in the output file which is also defined in the input file. In the present version, the electronic charge density, it integral over the whole space and polarization density (the difference from the initial values) are recorded in the output file.
