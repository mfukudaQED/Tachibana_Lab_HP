Platforms

QEDynamics is written in Fortran 90. This is designed to run on unix or linux systems.

  • Tested platforms
    1. Kyoto University Supercomputer system
      • OS : Red Hat Enterprise Linux 6
      • Compiler : Intel fortran compiler (ifort)
We expect that for other platforms QEDynamics is compiled and runs after small modification (see an installation document), though we have not tested.
